# To-Do Work University Project

React (Create React App) project with:
- Login screen
- Sidebar navigation (Home, To-Do, Notes, Calendar, About)
- Multi-language (Arabic / English)
- Dark / Light theme
- To-Do list with time, countdown, priority, search, filters, counters
- Notes section
- Calendar with per-day events
- About page describing the project
- Footer with developer name: Abdulrahman Sabha
- LocalStorage persistence for todos, notes, events, language, theme, view

## Run

npm install
npm start
